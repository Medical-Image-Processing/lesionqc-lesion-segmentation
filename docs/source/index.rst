lesionqc documentation
===================================================

lesionqc is a python package containing a variety of metrics to analyze the
quality of a lesion segmentation result.
This package was developed by `Jacob Reinhold`_ and the other students and researchers of IACL_.

.. _IACL: https://iacl.ece.jhu.edu/index.php/Main_Page
.. _`Jacob Reinhold`: https://jcreinhold.github.io
